# IMDB-Movie-Review-Sentiment-Analysis-IMRSA
### End-to-end sentiment analysis pipeline using LSTM, text preprocessing, and TensorFlow/Keras.

This project uses LSTM neural networks for IMDb sentiment analysis and is deployed using Streamlit.

 ---------------------------------
| Item       | Details            |
| ---------- | ------------------ |
| Model      | LSTM               |
| Dataset    | IMDb               |
| Framework  | TensorFlow/Keras   |
| Deployment | Streamlit          |
| Task       | Sentiment Analysis |
 ---------------------------------
